about spice2x

spice2x (spicetools2x) is a fork of SpiceTools, focused on addressing bugs, usability, and quality-of-life improvements

spice2x binaries are meant to be drop-in replacements for existing installs (spice.exe, spice64.exe, spicecfg.exe)

Efforts will be made to keep up to date with upstream changes.

known issues

For EG Valk mode, I/O emulation is incomplete. Please enable "Valkyrie Mode BIO2 (KFC) IO" or "Force BIO2 (KFC) IO in Valkyrie mode" patch.

When using the option(s) to automatically rotate display on launch, the game may start with the wrong refresh rate.

Some anti-virus software may (incorrectly) flag the binaries as malware. If you have concerns, you are free to inspect the included source files and compile on your own. As of 23-03-17-0731 build, UPX compression has been disabled. This results in a larger file size on disk, but hopefully gets incorrectly detected as malware less often. This is an unfortunate situation created by the fact that actual malware often use UPX for obfuscation, which feeds into various anti-virus software algorithms, which then causes other UPX-compressed software to be falsely flagged.
