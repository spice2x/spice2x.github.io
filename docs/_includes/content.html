about spice2x

spice2x is a fork of SpiceTools, focused on addressing bugs, usability, and quality-of-life improvements

spice2x binaries are meant to be drop-in replacements for existing installs (spice.exe, spice64.exe, spicecfg.exe).

Existing SpiceCompanion apps should also be compatible with spice2x: Google Play store, Apple App Store.

We are not associated with the original SpiceTools developers. SpiceCompanion apps are also separate projects owned by the app developers.

Efforts will be made to keep up to date with upstream changes.

beta downloads

Last updated on: April 3rd, 2023.

Note that while beta releases are mostly stable, there may be some bugs. If you experience any issues, please use the most recent stable release. Additionally, some of the new features and option flags may change. It is recommended that you back up the config file (%appdata%\spicetools.xml). These builds will be "promoted" to stable once they have been thoroughly tested.

  • spice2x-23-04-03
    • IIDX and SDVX subscreen overlays are now in a floating window. New options were added to control the behavior
    • Major effort to fix various touch input issues.
    • Known issue: SpiceCfg may fail to open in some cases, will be fixed in next release

known issues in latest release

Some anti-virus software may (incorrectly) flag the binaries as malware. If you have concerns, you are free to inspect the included source files and compile on your own.

EG may launch at wrong refresh rate when using the auto-rotate option. Please enable -9on12 and -graphics-force-refresh to work around this.

Hopefully fixed in the latest beta: using a touch screen for the game after interacting with Spice overlay may cause touches to be less responsive. As a workaround, use -overlaydisable option to turn off Spice overlay in game.
