about spice2x

spice2x is a continuation of SpiceTools project.

spice2x binaries are meant to be drop-in replacements for existing installs (spice.exe, spice64.exe, spicecfg.exe). For a list of features, check out this page.

notice - September 2025

  • If you got the latest IIDX update, please ensure you updated spice2x to the latest version, as it contains fixes to better support iidx32. Make sure you do NOT check -audiohookdisable.
  • If SDVX fails to launch with touch_effect error, it means you have bad data. This is not a tools issue. Click for details.

suggested addons

Check out these cool tools that can be used with spice:
  • SpiceCompanion SpiceCompanion - control spice from your phone / tablet / another PC. Archive only, not actively maintained by anyone.
  • ifs_layeredfs - for game asset modding
  • spicehue - control Philips Hue lights over spice API
  • slidershim - use your phone / tablet as keyboard input
